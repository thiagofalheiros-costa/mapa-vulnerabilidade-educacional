# Mapa de Vulnerabilidade Educacional

Projeto de Data Science aplicado a políticas públicas educacionais.

## Objetivo

Construir um índice municipal de vulnerabilidade educacional para
identificar e priorizar territórios com maiores desafios relacionados
ao fluxo escolar, à distorção idade-série, à infraestrutura e ao
contexto socioeconômico.

## MVP

- Unidade de análise: município;
- Recorte territorial: Rio Grande do Sul;
- Ano-base inicial: 2024;
- Redes analisadas: estadual e municipais;
- Etapa inicial: anos finais do Ensino Fundamental.

## Fontes de dados

- Censo Escolar — Inep;
- Indicadores Educacionais — Inep;
- Malha Municipal — IBGE;
- Códigos dos Municípios — IBGE.

## Estrutura

- `data/raw`: arquivos originais;
- `data/interim`: dados intermediários;
- `data/processed`: bases analíticas;
- `notebooks`: exploração e validação;
- `src`: código reutilizável;
- `reports`: mapas, tabelas e gráficos;
- `dashboard`: aplicação interativa;
- `docs`: documentação metodológica.
